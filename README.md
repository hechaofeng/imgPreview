# imgPreview
JS/Javascript plugin for imgPreView.js 
使用方法：
        1.在图片上加上class类‘img-view’;
        2.本插件基于jquery;
        3.在引入jquery后,引入imgView.min.js即可;
        4.本jsg=更适合移动端;
